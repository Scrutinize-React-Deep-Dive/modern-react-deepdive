## ✔️ 스터디 소개

## ✔️ 스터디 진행 방식

- 매주 금요일 낮 12시부터 1시까지 React 스터디 시간을 가집니다.
- 발표자는 랜덤으로 선정합니다.
- 수요일까지 스터디에 대한 내용을 정리해서 올립니다.
- 디스커션 보드에 각 챕터에 대한 질문을 남깁니다.
- 목요일과 금요일에는 풀 리퀘스트(Pull Request)를 받은 내용을 읽고, 코멘트를 남기며 답변합니다.
